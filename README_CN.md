Flow 是一个极简主义的番茄钟应用，旨在帮助你进入心流状态，实现深度工作。

## ✨ 功能特性

*   **专注计时**: 经典的 25 分钟专注模式，支持自定义时长。
*   **沉浸动效**: 呼吸般的背景与流光文字，助你平复心境。
*   **数据统计**: 详细的热力图记录你的每一次专注，见证点滴进步。
*   **标签系统**: 为任务分类（Reading, Coding, Meeting 等），清晰管理时间。
*   **极简界面**: 无干扰的暗色模式设计，完美适配 macOS。

## 📥 安装指南

1.  从 [Releases](https://github.com/MuQY1818/Flow/releases) 页面下载最新的 `.dmg` 文件。
2.  打开 `Flow.dmg`。
3.  将 `Flow.app` 拖入 `Applications` 文件夹。
4.  开启你的心流之旅！

## 🚀 使用方法

1.  **开始专注**: 点击 "Start Focus" 按钮，界面将泛起微光，提示你已进入心流模式。
2.  **切换模式**: 使用顶部标签栏在 "Controls"（计时）和 "Stats"（统计）之间切换。
3.  **选择标签**: 点击计时器上方的标签菜单，选择当前活动的类型。
4.  **设置**: 点击右下角的滑块图标，调整计时时长和声音设置。
